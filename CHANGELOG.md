# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2025-01-01

### Added
- Initial Magento 2 module boilerplate
- Claude Code integration (skills, rules, commands, agents, hooks)
- Module skeleton with standard directory structure
- PHPUnit test configuration with sample test
- PHPCS with Magento2 coding standard
- PHPStan static analysis (level 6)
- PHPMD mess detection with Magento-tuned rules
- PHP-CS-Fixer for auto-formatting
- CaptainHook for git hooks (pre-commit, commit-msg, pre-push)
- Makefile with developer commands
- Scripts: init.sh, check-setup.sh, run-tests.sh
- Optional Docker Compose setup (PHP, Nginx, MySQL, OpenSearch, Redis, Mailhog)
- Documentation: README, CONTRIBUTING, CHANGELOG, CLAUDE.md
